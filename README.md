# Apex Academy Website

## Project overview
Apex Academy is a responsive educational website designed to prove a smooth user experience.

## Features
- User Signup and Login pages
- Local storage Authentication
- Responsive Design
- clean UI/UX

## Technologies used
- HTML
- CSS
- JavaScript

## project structure
- index.html : Home page with multiple sections(faculty, Admission, location, Online learning, Student Testimonials sections)
- login.html : Login page
- signup.html : Signup page
- style.css : styling of html pages
- script.js : functionality of login and signup pages
- images folder : All images related to website
- ProjectReport.pdf : Apex Academy project report

## Challenges Faced
- Handling login validation using localStorage
- Fixing invalid login issuses
